task_mng - automated task distribution system.

--------
Programming language: Erlang

Compilation and testing: rebar2 or erlang.mk

--------
### Building via erlang.mk:
```
$ git clone https://github.com/alina-tushych/task_mng.git
$ cd task_mng
$ make
$ ./start.sh
```
### Building via rebar2:
```
$ git clone https://github.com/alina-tushych/task_mng.git
$ cd task_mng
$ rebar get
$ rebar co
$ ./start.sh
```
